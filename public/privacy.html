Политика конфиденциальности

Последнее обновление: 13 июня 2026 г.

1. Кто мы

Splitaway — приложение для совместного учёта расходов в поездках. Разработано как личный проект (rafaelbabaew@googlemail.com). Приложение работает как прогрессивное веб-приложение (PWA) и доступно по адресу splitaway.netlify.app.

2. Какие данные мы собираем

Splitaway собирает только данные, которые вы вводите сами:

  • Данные поездки — название, направление, список участников (имена, без привязки к реальным личностям).
  • Финансовые записи — суммы расходов, платёжные распределения. Мы не обрабатываем реальные платежи.
  • Push-подписки — endpoint и ключи уведомлений для отправки оповещений (только с вашего явного согласия).

Мы не собираем: имена пользователей, email, телефон, геолокацию, ID устройства, рекламные идентификаторы, cookies для трекинга.

3. Как мы храним данные

  • Локально — данные сохраняются в localStorage вашего браузера.
  • В облаке — при синхронизации данные отправляются в Supabase (PostgreSQL). Доступ защищён токенами приглашения и Row Level Security.
  • Хостинг — статика хостится на Netlify (США). Данные Supabase хранятся в регионе EU West.

4. Совместный доступ к поездке

Поездка доступна только тем, у кого есть ссылка-приглашение с уникальным токеном. Данные не открыты публично. Организатор поездки может удалить все данные в любой момент через настройки.

5. Сторонние сервисы

  • Supabase — база данных и реалтайм (политика).
  • Netlify — хостинг (политика).
  • Anthropic Claude — OCR чеков через API прокси на Netlify Functions (политика). Изображения чеков не сохраняются.

6. Удаление данных

Чтобы удалить все данные поездки: откройте приложение → Настройки → «Удалить поездку». Это необратимо удалит все записи из базы данных. Данные в localStorage очищаются через «Настройки браузера → Очистить данные сайта».

7. Дети

Приложение не предназначено для детей до 13 лет и не собирает данные о них намеренно.

8. Изменения

При существенных изменениях политики мы обновим дату в заголовке. Актуальная версия всегда доступна по адресу splitaway.netlify.app/privacy.html.
